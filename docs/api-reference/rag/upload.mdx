---
title: Upload Document
api: POST /v1/documents/upload
description: Upload a text document to be chunked and stored as vectors in a collection.
---

<ParamField header="x-api-key" type="string" required>
  API key with `admin` or `readwrite` role.
</ParamField>

<ParamField body="collection_name" type="string" required>
  Target collection name. The collection must already exist.
</ParamField>

<ParamField body="file" type="file" required>
  The document to upload. Currently only `.txt` files are supported. Sent as multipart form data.
</ParamField>

<ParamField body="include_timing" type="boolean" default="false">
  When `true`, the response includes a `timing_ms` object with `embedding_ms`, `storage_ms`, and `total_ms` breakdowns.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST http://localhost:8000/v1/documents/upload \
  -H "x-api-key: your-key" \
  -F "collection_name=articles" \
  -F "file=@/path/to/document.txt"
```

```bash cURL (with timing)
curl -X POST http://localhost:8000/v1/documents/upload \
  -H "x-api-key: your-key" \
  -F "collection_name=articles" \
  -F "file=@/path/to/document.txt" \
  -F "include_timing=true"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "status": "success",
  "data": {
    "document_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "chunks_created": 12
  },
  "error": null
}
```

```json With timing
{
  "status": "success",
  "data": {
    "document_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "chunks_created": 12,
    "timing_ms": {
      "embedding_ms": 145.3,
      "storage_ms": 18.7,
      "total_ms": 164.0
    }
  },
  "error": null
}
```
</ResponseExample>

<Note>
  The document is split into chunks, each chunk is embedded and stored as a vector in the specified collection. The `document_id` is a UUID assigned to the upload, and `chunks_created` indicates how many vector chunks were generated from the file.
</Note>
