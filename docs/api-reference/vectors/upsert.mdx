---
title: Upsert Vector
api: POST /v1/collections/{collection_name}/upsert
description: Insert or update a single vector. If the external_id already exists, it is updated.
---

<ParamField path="collection_name" type="string" required>
  Target collection name.
</ParamField>

<ParamField header="x-api-key" type="string" required>
  API key with `admin` or `readwrite` role.
</ParamField>

<ParamField body="external_id" type="string" required>
  Unique identifier for the vector within this collection. If it already exists, the vector and metadata are overwritten.
</ParamField>

<ParamField body="vector" type="number[]">
  Float array. Length must exactly match the collection's `dim`. Either `vector` or `text` must be provided. If both are given, `vector` takes precedence.
</ParamField>

<ParamField body="text" type="string">
  Plain text to embed. The backend generates a vector using the configured embedding model. Either `text` or `vector` must be provided. If both are given, `vector` takes precedence.
</ParamField>

<ParamField body="metadata" type="object">
  Optional JSON object. Attach any structured data. Max size: 10KB.
</ParamField>

<ParamField body="include_timing" type="boolean" default="false">
  When `true`, the response includes a `timing_ms` object with `embedding_ms`, `storage_ms`, and `total_ms` breakdowns.
</ParamField>

<RequestExample>
```bash cURL (with vector)
curl -X POST http://localhost:8000/v1/collections/articles/upsert \
  -H "x-api-key: your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "external_id": "doc-1",
    "vector": [0.1, 0.2, 0.3],
    "metadata": {
      "title": "Getting started",
      "author": "Alice",
      "tags": ["tutorial"]
    }
  }'
```

```bash cURL (with text)
curl -X POST http://localhost:8000/v1/collections/articles/upsert \
  -H "x-api-key: your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "external_id": "doc-1",
    "text": "Getting started with vector databases",
    "metadata": {
      "title": "Getting started",
      "author": "Alice"
    },
    "include_timing": true
  }'
```
</RequestExample>

<ResponseExample>
```json Inserted
{
  "status": "success",
  "data": {
    "external_id": "doc-1",
    "status": "inserted"
  },
  "error": null
}
```

```json Updated
{
  "status": "success",
  "data": {
    "external_id": "doc-1",
    "status": "updated"
  },
  "error": null
}
```

```json With timing
{
  "status": "success",
  "data": {
    "external_id": "doc-1",
    "status": "inserted",
    "timing_ms": {
      "embedding_ms": 12.4,
      "storage_ms": 3.1,
      "total_ms": 15.5
    }
  },
  "error": null
}
```

```json Dimension Mismatch (400)
{
  "status": "error",
  "data": null,
  "error": {
    "code": 400,
    "message": "Dimension mismatch: expected 384, got 3"
  }
}
```
</ResponseExample>
